üìö Quick Integration Guide

To integrate Passive CAPTCHA into your website:

  1. Generate your script token in the admin dashboard
  2. Add the script to your website's <head> section:
<script> (function() { var script = document.createElement('script'); script.src = 'YOUR_API_ENDPOINT/api/script/generate?token=YOUR_SCRIPT_TOKEN'; script.async = true; script.defer = true; document.head.appendChild(script); })(); </script>

That's it! The script will automatically start collecting behavioral data and provide verification results.
